# studhouse-front

## Getting Started
First, install the npm version >= v18.17.0
npm i

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
